Bedanya SHA256 vs Pepper:
• SHA256 = algoritma hash (ngubah text jadi 64 hex).
• Pepper = secret tambahan di server. Jadi deviceKey = SHA256(deviceId + "|" + pepper).
Tanpa pepper, orang bisa nebak / bikin hash sendiri. Dengan pepper, harus tau pepper dulu.
